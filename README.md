# Western Ghats Biodiversity Portal 🌿

A web platform dedicated to exploring the biodiversity, wildlife, and conservation efforts of the Western Ghats.
This project aims to provide structured information about national parks, wildlife sanctuaries, species, and conservation initiatives in the region.

---

## 🌏 About the Project

The Western Ghats is one of the world's most important biodiversity hotspots.
This platform provides information about:

* National Parks
* Wildlife Sanctuaries
* Species and biodiversity
* Conservation projects
* Articles and research
* Photo galleries of wildlife and landscapes

Example protected areas included in the platform:

* Silent Valley National Park
* Periyar National Park
* Nagarhole National Park

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/your-username/western-ghats.git
```

Move into the project folder:

```
cd western-ghats
```

Create virtual environment:

```
python -m venv venv
```

Activate the environment:

Windows

```
venv\Scripts\activate
```

Linux / Mac

```
source venv/bin/activate
```

Install dependencies:

```
pip install -r requirements.txt
```

Run migrations:

```
python manage.py migrate
```

Start the development server:

```
python manage.py runserver
```

Open in browser:

```
http://127.0.0.1:8000
```
